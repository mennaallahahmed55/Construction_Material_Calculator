# Construction Material Calculator

A simple WPF desktop application for calculating and managing construction material data.

## Features

* Add construction material data
* Calculate material quantities
* Manage material records
* Simple desktop user interface
* Local data storage

## Tech Stack

C# • WPF • MVVM • .NET

## Project Structure

* WPF desktop application
* Clean and simple UI
* Organized project files

## Purpose

This project was built as part of my learning journey in AEC software development, focusing on creating practical tools for engineering and construction workflows.
